# Python_analysis
Store Sales Analysis with Python
# Висновки

## Загальна ефективність
Аналіз показав, що основними драйверами доходу компанії є три категорії товарів: **Office supplies**, **Clothes** та **Beverages**. Вони забезпечують найвищу прибутковість. Категорія **Household** демонструє негативну рентабельність і потребує додаткового розслідування причин збитковості.

## Канали збуту
Суттєвої різниці між онлайн- та офлайн-продажами за обсягами та прибутковістю не виявлено — обидва канали працюють приблизно рівно. Це може свідчити про:
- добре збалансовану стратегію каналів;
- або приховані відмінності в підсегментах (категорії товарів, сегменти клієнтів).

## Географія
Європейський регіон істотно перевищує Азійський за обсягами продажів.  
**Топ-3 країни за кількістю продажів:** Македонія, Сан-Маріно, Андорра.  
**Найнижчі показники:** Албанія, Хорватія, Монако.  
Присутність малих країн у топі може вказувати на локальні особливості (акції, партнерства, сезонність).

## Логістика та час доставки
- Найдовший середній час доставки: **Cereal**, **Baby food**.  
- Найшвидша доставка: **Personal Care**, **Clothes**.  
- За країнами: найшвидше — **Croatia**, найповільніше — **Hungary**.  
Це свідчить про можливі логістичні вузькі місця або різницю в локальних операціях.

## Взаємозв’язок часу доставки і продажів
Прямої кореляції між середнім часом доставки та обсягом продажів не виявлено. Це означає, що для покупців важливішими є інші фактори:
- ціна,
- асортимент,
- бренд.  

Можливо, вплив часу доставки проявляється в інших метриках (повторні покупки, задоволеність клієнтів).
